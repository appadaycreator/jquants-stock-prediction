jQuants IDトークン管理

トークンの再発行と管理を行います

🔐 jQuants IDトークン再発行

💡 注意事項:

  • トークンは24時間で期限切れになります
  • 認証情報は安全に管理してください
  • 再発行後はアプリケーションを再起動してください

📋 使用方法

1. 手動でのトークン再発行

上記のフォームを使用して、jQuantsの認証情報を入力し、トークンを再発行できます。

2. コマンドラインでの再発行

# 仮想環境をアクティベート

source venv_token/bin/activate

# トークン再発行スクリプトを実行

python3 reissue_jquants_token.py

3. 環境変数の設定

再発行後、以下の環境変数が自動的に設定されます：

  • JQUANTS_EMAIL - メールアドレス
  • JQUANTS_PASSWORD - パスワード
  • JQUANTS_ID_TOKEN - IDトークン
⚠️

セキュリティに関する注意

  • • 認証情報は安全に管理してください
  • • .envファイルをGitにコミットしないでください
  • • トークンは24時間で期限切れになります
  • • 定期的なトークン更新を推奨します
